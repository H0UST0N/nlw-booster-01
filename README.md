# Ecoleta

Projeto para cadastramento e localização de pontos de coleta de lixo reciclável desenvolvido durante a Next Level Week #01

## Tecnologias

Este projeto foi desenvolvido com:

* Node.js

* React

* ReactNative

## Utilização

Clone o repositorio

* server (Api)
npm install
npm run dev

* web (Aplicação Web)
npm install
npm start

* mobile (Aplicação Mobile)
npm install
npm start
